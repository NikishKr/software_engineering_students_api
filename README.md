# Students API

---
## Описание
Fastapi проект, предоставляющий API для сохранения и получении информации о студентах и группах студентов.

---
## Требования
- Python 3.12
- Docker Compose
- Библиотеки, указанные в файле `requirements.txt`

---
## Установка
1. Скачайте проект
2. Скопируйте содержимое из `.env.example` в `.env`
3. Скачайте все необходимые библиотеки

---
## Использование
1. Запустите базу данных с помощью `docker-compose up -d`
2. Запустите сервер с помощью `uvicorn app.main:app --reload`
3. Чтобы протестировать приложение, воспользуйтесь:
   - API: http://localhost:8000/
   - Docs: http://localhost:8000/docs
4. По окончании работы завершите работу сервера и очистите базу данных: `docker-compose down -v`

---
## Эндпоинты API
### Корень
- /api/v1/
### Студенты
- POST /students/ - создать студента
- GET /students/ - получить список всех студентов
- GET /students/{id} - получить информацию о студенте по ID
- DELETE /students/{id} - удалить студента
- POST /students/{id}/assign_group - добавить студента в группу
- DELETE /students/{id}/remove_group - удалить студента из группы
- PUT /students/{id}/transfer - перевести студента в другую группу
### Группы
- POST /groups/ - создать группу
- GET /groups/ - получить список всех групп
- GET /groups/{id} - получить информацию о группе по ID
- DELETE /groups/{id} - удалить группу
- GET /groups/{id}/students - получить список студентов группы
